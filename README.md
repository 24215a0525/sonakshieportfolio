Sonakshi Portfolio Website
📌 Overview

This is my personal developer portfolio website that showcases my projects, skills, and achievements.
The website highlights my work in full-stack web development and includes projects such as Hostel Finding and Booking System and Smart Sound – AI Emotion-Based Music Recommendation System.

The portfolio allows visitors to explore my projects, view technologies used, and contact me.
